# rails-template

Rails 项目模板

## 如何使用


* 新建项目
```shell
rails new myapp -m https://raw.github.com/work-design/template/master/template.rb
```

* 已存在的项目
```shell
bin/rails app:template LOCATION=https://raw.github.com/work-design/template/master/better.rb
```
